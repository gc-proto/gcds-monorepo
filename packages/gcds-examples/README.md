# GCDS Examples Package

This is a placeholder directory for the `gcds-examples` fork.

**To set up for development:**

1. Remove this placeholder directory:
   ```bash
   rm -rf packages/gcds-examples
   ```

2. Clone the actual fork repository:
   ```bash
   git clone https://github.com/gc-proto/gcds-examples.git packages/gcds-examples
   ```

3. Install dependencies:
   ```bash
   pnpm install
   ```

See [SETUP.md](../../SETUP.md) for complete setup instructions.
